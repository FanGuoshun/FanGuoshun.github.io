Ajax介绍

Posted on 2019-07-27

  • 概念：Asynchronous JavaScript And XML 异步的JavaScript和XML

    • 异步和同步：客户端和服务器端相互通信的基础上
      • 同步：客户端必须等待服务器端的响应。在等待的期间客户端不能做其它操作。
      • 异步：客户端不需要等待服务器端的响应。在服务器处理请求的过程中，客户端可以进行其它的操作。
    • Ajax是一种无需重新加载整个网页的情况下，能够更新部分网页的技术。

两行代码解决命令行启动python脚本找不到包问题

Posted on 2019-07-22

  • 找到出现Import Error的脚本，写上两行代码（写在第一行）：

    import sys
sys.path.append("/home/aaron/project/python2_env/env/lib/python2.7/site-packages")

  • 即将自己电脑上的python虚拟环境下的存放各种包的文件夹加入sys.path。因为Linux命令行启动python脚本只会从当前目录寻找依赖的包，所以要手动添加，防止报错。

ChainSpace合约执行过程

Posted on 2019-07-21 Edited on 2019-07-27

  • ChainSpace中的合约大概分为三个部分：init(),method(),checker()

    <!--more-->

    • init():初始化合约，要在函数名上加上装饰器@contract.method(init)

      # 初始化合約
@contract.method('init')
def init():
    # return
    return {
        'outputs': (dumps({'type': 'BankToken'}),),
    }

    • method():合约里的功能函数，比如转账功能，投票功能等。同样在其函数名上加上装饰器@contract.method(函数名）

      # 执行转账交易
@contract.method('auth_transfer')
def auth_transfer(inputs, reference_inputs, parameters, priv):
    # compute outputs
    amount = loads(parameters[0])  # 转账金额
    new_from_account = loads(inputs[0])  # 旧账户
    new_to_account = loads(inputs[1])  # 新账户
    new_from_account["balance"] -= amount  # 更新旧账户余额
    new_to_account["balance"] += amount   # 更新新账户余额

    # hash message to sign
    hasher = sha256()
    hasher.update(dumps(inputs).encode('utf8'))
    hasher.update(dumps(reference_inputs).encode('utf8'))
    hasher.update(dumps(parameters[0]).encode('utf8'))
    #  生成hash摘要

    # sign message
    G = setup()[0]
    sig = do_ecdsa_sign(G, unpack(priv), hasher.digest())
    #   生成签名
    # return
    return {
        'outputs': (dumps(new_from_account), dumps(new_to_account)),
        'extra_parameters': (pack(sig),)
    }

    • checker():执行验证功能，验证合约的执行是否正确。函数名上加上装饰器@contract.checker(函数名)。该种函数只返回True或false

      #   验证转账交易是否合法
@contract.checker('auth_transfer')
def auth_transfer_checker(inputs, reference_inputs, parameters, outputs, returns, dependencies):
    try:
        amount = loads(parameters[0])
        input_from_account = loads(inputs[0])
        input_to_account = loads(inputs[1])
        output_from_account = loads(outputs[0])
        output_to_account = loads(outputs[1])
        sig = unpack(parameters[1])

        # check format
        if len(inputs) != 2 or len(reference_inputs) != 0 or len(outputs) != 2 or len(returns) != 0:
            return False
        if input_from_account['pub'] != output_from_account['pub'] or input_to_account['pub'] != output_to_account[
            'pub']:
            return False

        # check tokens
        if input_from_account['type'] != 'BankAccount' or input_to_account['type'] != 'BankAccount':
            return False
        if output_from_account['type'] != 'BankAccount' or output_to_account['type'] != 'BankAccount':
            return False

        # amount transfered should be non-negative
        if amount < 0:
            return False

        # amount transfered should not exceed balance
        if input_from_account['balance'] < amount:
            return False

        # consistency between inputs and outputs
        if input_from_account['balance'] != output_from_account['balance'] + amount:
            return False
        if input_to_account['balance'] != output_to_account['balance'] - amount:
            return False

        # hash message to verify signature
        hasher = sha256()
        hasher.update(dumps(inputs).encode('utf8'))
        hasher.update(dumps(reference_inputs).encode('utf8'))
        hasher.update(dumps(parameters[0]).encode('utf8'))

        # recompose signed digest
        pub = unpack(input_from_account['pub'])

        # verify signature
        (G, _, _, _) = setup()
        return do_ecdsa_verify(G, pub, sig, hasher.digest())

        return True

    except (KeyError, Exception):
        return False

    • 调用合约的用户在本地执行完合约上的相关功能后，将执行结果以及输入通过post请求发送到server端（post请求体中包含method()的输入和输出。）

      if __debug__ and not _checker_mode.on:
        # Automatically run the checker to ensure procedure output passes check.
        if method_name not in self.checkers and method_name != "init":
            print("POTENTIAL ERROR: '%s' method has no checker." % method_name)
        elif method_name != "init":
            txt = transaction_inline_objects(return_value)

            app = self.flask_app.test_client()

            result = json.loads(args.data)["success"]

    • server端收到请求后，调用合约里的checker对合约执行结果进行检查。（默认端口号为5000)

      @self.flask_app.route('/' + self.contract_name + '/' + method_name, methods=['POST'], endpoint=method_name)
def checker_request():
    dependencies = (request.json['dependencies'] if 'dependencies' in request.json else ())
    for dependency in dependencies:
        dependency['inputs'] = tuple(dependency['inputs'])
        dependency['referenceInputs'] = tuple(dependency['referenceInputs'])
        dependency['outputs'] = tuple(dependency['outputs'])
        dependency['parameters'] = tuple(dependency['parameters'])
        dependency['returns'] = tuple(dependency['returns'])

    return function_wrapper(
        tuple(request.json['inputs']),
        tuple(request.json['referenceInputs']),
        tuple(request.json['parameters']),
        tuple(request.json['outputs']),
        tuple(request.json['returns']),
        dependencies
    )
