Ajax介绍

Posted on 2019-07-27

  • 概念：Asynchronous JavaScript And XML 异步的JavaScript和XML

    • 异步和同步：客户端和服务器端相互通信的基础上
      • 同步：客户端必须等待服务器端的响应。在等待的期间客户端不能做其它操作。
      • 异步：客户端不需要等待服务器端的响应。在服务器处理请求的过程中，客户端可以进行其它的操作。
    • Ajax是一种无需重新加载整个网页的情况下，能够更新部分网页的技术。

  2. 找到.cargo 文件夹，进入.cargo 当前目录，在当前目下创建 config 文件 ，见下图，打开 config 文件，编写以下内容：

两行代码解决命令行启动python脚本找不到包问题

Posted on 2019-07-22

  • 找到出现Import Error的脚本，写上两行代码（写在第一行）：

    import sys
sys.path.append("/home/aaron/project/python2_env/env/lib/python2.7/site-packages")

  • 即将自己电脑上的python虚拟环境下的存放各种包的文件夹加入sys.path。因为Linux命令行启动python脚本只会从当前目录寻找依赖的包，所以要手动添加，防止报错。
