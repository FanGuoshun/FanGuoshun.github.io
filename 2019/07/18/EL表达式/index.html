EL表达式

Posted on 2019-07-18

  • EL表达式:

    • 概念：Expression Language
    • 作用：替换和简化JSP中Java代码书写
    • 语法：${表达式}
    • 注意：
      • jsp默认支持el表达式。如果要忽略el表达式：
        • 设置jsp中page指令中：isELIgnored=”true” 忽略当前页面所有EL表达式
        • ${表达式}: 忽略这个el表达式

  • 使用：

    • 运算：
      • 运算符：
        • 算数运算符
        • 比较运算符
        • 逻辑运算符
        • 空运算符 empty
          • 判断字符串、集合、数组对象是否为NULL并且长度是否为0
          • ${empty list}
    • 获取值：
      • 只能从域对象中获取值
      • 语法：
        • ${域名称,键名}:从指定域中获取键值
          • 域名称：
            • pageScope –> pageContext
            • requestScope –> request
            • sessionScope –> session
            • applicationScope –> application
          • 举例：在request域中存储了name=张三
          • 获取：${requestScope.name}
        • ${键名}:标识依次从最小的域中查找是否有该键对应的值，直到找到为止。
          • 获取对象List集合、Map集合的值
          • 对象：${域名称.键名.属性名}
            • 实际上是调用对象的getter方法
          • List集合：${域名称.键名[索引]}
            • 若发生数组越界，服务器不会报错，页面显示空字符串
          • Map集合：
            • ${域名称.键名.key名称}
            • ${域名称.键名[“key名称”]}

  • 隐式对象：

    • el表达式中有11个隐式对象
    • pageContext:
      • 获取jsp中其它八个内置对象
        • ${pageContext.request.contextPath}:动态获取虚拟目录
